🛠 ADHD — Service Desk for University Department
ADHD (Alert Department Help Desk) — це мікросервісна система для автоматизації подання та обробки заявок на технічне обслуговування кафедр. Система дозволяє студентам та викладачам швидко повідомляти про несправності, а технічному персоналу — оперативно керувати заявками через Telegram.

🌟 Основні можливості
Web-платформа: Подання заявок без реєстрації через інтуїтивну форму.

Публічний моніторинг: Таблиця статусів у реальному часі (Нова, В роботі, Виконано, Відхилено).

Telegram Bot Integration: Миттєві сповіщення для лаборантів з інтерактивними кнопками управління.

Concurrency Control: Захист від "Race Condition" (одна заявка не може бути прийнята двома виконавцями одночасно).

Containerization: Повне розгортання всієї інфраструктури однією командою через Docker.

🏗 Архітектура системи
Проект побудований на базі мікросервісів, що забезпечує незалежність модулів:

Backend API (.NET 10): Центральний вузол, обробка бізнес-логіки.

Database (PostgreSQL): Надійне збереження даних та станів.

Telegram Bot (Python/aiogram): Інтерфейс для виконавців.

Frontend (Vanilla JS): Легкий та швидкий клієнтський інтерфейс.

Reverse Proxy (ngrok): Для створення публічного тунелю до локального сервера.

🛠 Технологічний стек
Backend: C# / .NET 10, Entity Framework Core.

Frontend: HTML5, CSS3 (Flexbox/Grid), JavaScript (Fetch API).

Bot: Python 3.11+, aiogram 3.x.

Database: PostgreSQL.

DevOps: Docker, Docker Compose, ngrok.

🚀 Швидкий старт (Deployment)
1. Клонування репозиторію
Bash
git clone https://github.com/your-username/adhd-project.git
cd adhd-project
2. Запуск через Docker Compose
Система автоматично змонтує базу даних, запустить API та бот:

Bash
docker-compose up --build -d
3. Налаштування зовнішнього доступу (ngrok)
Для взаємодії з Telegram API на локальній машині:

Bash
ngrok http 8080
Не забудьте оновити Webhook URL у налаштуваннях бота на отриману адресу від ngrok.

📋 Статуси заявок
🟡 Нова: Заявка створена, очікує на розгляд.

🔵 В роботі: Виконавець прийняв задачу в Telegram.

🟢 Виконано: Проблему усунено.

🔴 Відхилено: Заявку скасовано (спам або некоректні дані).
